## Convite de Cadastro
## Dados
```json
{
    "id": "string",
    "email": "user@example.com",
    "name": "string",
    "createdAt": 0,
    "expiresAt": 0,
    "status": "finished"
  }
```

## Requisitos
- ❎ Um usuário master ou administrador deve conseguir visualizar os convites que ele enviou.
    - ❎ Deve retornar uma lista de convites com o **estado**, podendo ser **finished**, **pending**.
